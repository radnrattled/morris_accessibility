* Clone the repo
* git clone https://github.com/radnrattled/morris_m_chatapp
* Open the folder 
* launch your MAMP (WAMP)
* Move folder into your htdocs(www) folder in MAMP(WAMP)
 * The app should open in your browser window using localhost.
